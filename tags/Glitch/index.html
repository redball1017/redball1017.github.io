make-oneindex-by-Glitch
2021年06月12日 231 字 大概 1 分钟

材料

1.一个Github账号

2.一个Glitch账号

3.一个Cloudflare账号

4.一个域名

正文

第一步

首先打开Glitch官网。然后登录你的Glitch账号。然后点击右上角头像旁边的”New Project”。然后再点击Import From Github

然后在弹出的对话框中输入”https://github.com/qkqpttgf/OneManager-php"然后点击确定

若你想自定义域名，你需要先点击左下角的Tools，然后在弹出的方框内选择”Custom Domains”

然后在输入框内输入你想要的域名然后点击下面的Add Domains

然后到你的DNS提供商那里添加CNAME记录，值填glitch.edgeapp.net

第二步

首先我们要获取网址。（如果是已经自定义完网址的请忽略）。首先我们要点及左上角的Share。在弹出的窗口中复制Live site，这个就是网址了。
